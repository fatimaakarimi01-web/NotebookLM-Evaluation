# NotebookLM Evaluation – BUS5001 Cloud Platforms and Analytics

This repository contains evidence from the evaluation of Google’s **NotebookLM** as part of Assignment 3, Question 4.  
The purpose of the experiment was to assess how NotebookLM can support common academic activities such as summarising lecture notes, generating study aids, and assisting with research tasks.

---

## Features Tested

The following NotebookLM features were evaluated using Week 04 Cloud Security & Governance lecture notes:

| Feature | Description | Screenshot |
|---------|-------------|------------|
| AI-generated Summary | NotebookLM produced a structured overview of key topics from the lecture notes. | `summary.png` |
| Q&A Assist | AI answered follow-up academic questions based on uploaded material. | `Q&A.png` |
| Flashcards | Automated flashcards were generated for revision purposes. | `flashcards.png` |
| Mind Map | NotebookLM converted the lecture content into a visual knowledge map. | `mindmap.png` |
| Podcast Mode | The tool converted the notes into an audio explanation for multitasking study. | `podcast.png` |

All evidence files are stored in this repository as PNG images.
